# LISTAS

No CSS, é possível personalizar listas, sejam elas ordenadas (```ol```), não ordenadas (```ul```) ou descritivas (```dl```), por meio de seus seletores. Além das propriedades específicas de listas, todas as propriedades de texto também podem ser aplicadas.

* ```list-style-type```: define o tipo de marcador;
    * ```disc```, ```circle```, ```square```, ```none```, entre outros.
* ```list-style-position```: define a posição do marcador;
    * ```inside```: dentro da área de conteúdo da lista;
    * ```outside```: fora da área de conteúdo (padrão).
* ```margin```: define as margens da lista;
* ````padding````: define o espaço interno da lista;
* ```list-style-image```: permite usar uma imagem como marcador.

As propriedades de marcador podem ser abreviadas com ```list-style```.

```css
ul, ol, dl {
    list-style-type: circle;
    list-style-position: inside;
    margin-left: 10px;
    padding: 10px;

    list-style: circle inside;
}

dt {
    font-weight: bold;
    font-size: 12px;
}

dd {
    font-size: 10px;
    padding: 8px;
}

```
