# LINKS

Os links são fundamentais para a navegação em sites. Com CSS, é possível personalizá-los visualmente e também controlar seus estados de interação, melhorando a experiência do usuário e a acessibilidade.

## Propriedades principais para links

As propriedades abaixo são as mais utilizadas para estilizar links:

* ```color```: define cor do texto do link;
* ```text-decoration```: remove ou adiciona sublinhados, podendo escolher a cor diferente ou não;
* ```font-family```: define a fonte do texto;
* ```font-size```: define o tamanho da fonte;
* ```background-color```: cor de fundo do link (usada para criar efeito de botão);
* ```padding```: 
* ```cursor```: define o tipo de cursor ao passar o mouse (pointer).

```css
a {
    color: blue;
    text-decoration: none;
    font-family: Arial, sans-serif;
    font-size: 14px;
    cursor: pointer;
}
```

## Ações com links

Os links podem ser estilizados de acordo com a interação do usuário, através do seletor dois pontos (:), como pode ser visto abaixo:

* ```:hover``` - mouse sobre o link;
* ```:link``` - link não visitado;
* ```:active``` - link sendo cliclado;
* ```:visited``` - link visitado;
* ```:focus``` - link focado (selecionado).

A ordem indicada num arquivo css é ```:link```, ```:visited```, ```:hover``` e ```:active```.

```css
a:link {
    color: blue;
    text-decoration: underline blue;
}

a:visited {
    color: purple;
}
```
