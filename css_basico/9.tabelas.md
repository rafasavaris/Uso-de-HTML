# TABELAS

É possível personalizar tabelas de diversas formas com CSS, aplicando estilos diretamente aos seletores relacionados à estrutura da tabela (```table```, ```tr```, ```th```, ```td```, etc.). Além das propriedades específicas, todas as propriedades de texto também podem ser utilizadas.

As principais propriedades são:

* ```border-collapse```: define se as bordas das células ficam juntas ou separadas:
    * ```collapse```: une as bordas;
    * ```separate```: mantém as bordas separadas.
* ```border-spacing```: define o espaço entre as bordas das células (usado com ```separete```);
* ```border```: define a borda das células;
* ```text-align```: define o alinhamento do texto (```left```, ```center``` ou ```right```);
* ```padding```: define o espaço interno das células.
* ```width```: define o quanto a tabela deve ocupar do espaço disponível;
* ```vertical-align```: alinhamento vertical do conteúdo (```top```, ```middle``` ou ```bottom```);
* ```background-color```: cor de fundo da tabela ou das células.

> Quando uma propriedade é aplicada ao seletor ```table```, ela afeta apenas a estrutura externa da tabela. Para estilizar as células internas, é necessário aplicar regras diretamente a ```th``` e ```td```.

É possível aplicar um estilo diferente às linhas pares ou ímpares de uma tabela, facilitando a leitura e a visualização dos dados.
Para isso, utiliza-se a pseudo-classe ```:nth-child()```, que seleciona elementos com base na sua posição em relação ao elemento pai. O valor informado entre parênteses pode ser:

* ```even```: linhas pares;
* ```odd```: linhas ímpares.

Vale ressaltar que o CSS inicia a contagem a partir de 1, e não de 0.

```css
table {
  border-collapse: collapse;
  width: 100%;
}

tr:nth-child(even) {
    background-color: #ccc
}

th {
  font-weight: bold;
  border: 1px solid black;
  text-align: center;
  padding: 15px;
  background-color: #f2f2f2;
}

td {
  border: 1px solid black;
  padding: 15px;
  text-align: left;
}
```
