Tags de texto em HTML

1) Tags de header

Tags de header em HTML são usadas para definir títulos e subtítulos em uma página, organizando o conteúdo de forma hierárquica e melhorando a semântica e a acessibilidade do site. Cabe ressaltar que elas não são autofechantes. Elas podem variar de <h1/> à <h6/>. O texto que deve aparecer como títulos ou subtítulos é escrito entre a abertura e o fechamento da tag. A tag <h1/> é indicada ser usada só uma vez, pois deve ser o título. Abaixo, estão exemplos dessas tags.

2) Parágrafos em HTML

A tag <p> é utilizada para definir parágrafos em HTML. Ela delimita blocos de texto, separando visualmente e semanticamente as ideias no conteúdo da página. Todo texto colocado entre <p> e </p> será exibido como um novo parágrafo, com espaçamento automático antes e depois. É uma das tags mais usadas para estruturar textos em páginas web. Apenas reforçando que, se desejado quebrar uma linha, deve ser usada a tag <br>.

3) Principais tags de formatação

As tags de formatação em HTML são usadas para alterar a aparência e o estilo do texto, sem necessariamente mudar sua estrutura semântica. Abaixo, são apresentadas algumas dessas tags, sendo que nenhuma é autofechante.
Tag <b> — Serve para colocar o texto em negrito
Tag <strong> — Também coloca o texto em negrito, mas dá mais ênfase em buscas em aplicativos de textos/internet
Tag <i> — Coloca o texto em itálico
Tag <i> — Coloca o texto em itálico.
Tag <em> — Também coloca o texto em itálico, mas com ênfase para aplicativos de texto/internet
Tag <mark> — Marca o texto como um marca-texto
Tag <small> — Diminui o tamanho do texto
Tag <sup> — Coloca o texto superior, dessa forma
Tag <sub> — Coloca o texto inferior, assim
Tag <del> — "Risca" o texto, dessa maneira
Tag <ins> — Define um texto que foi inserido num documento. Normalmente, o navegador sublinha, assim: texto
Tag <blockquote> — Define uma seção citada de outra fonte. Os navegadores geralmente recuam esses elementos, além de que eles são elementos independentes, usados fora de paragráfos, como em seguida:

Às vezes a vida é um filme e eu sou uma atriz no Vale do Silício. Rimando como se a minha vida dependesse disso. - Extraordinária, Ebony, 2025.

Tag <q> — Define uma citação curta. Navegadores normalmente inserem aspas ao redor, como visto em seguida:
Me sinto extraordinária, dona de uma mente brilhante. - Extraordinária, Ebony - 2025
Tag <address> — Define as informações de contato do autor/proprietário de um documento ou artigo. O texto no elemento geralmente é renderizado em itálico, e os navegadores sempre adicionam uma quebra de linha antes e depois desse elemento. Tag <address> — Define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura etc.). O texto geralmente é renderizado em itálico. Tag <bdo> — BDO significa substituição bidirecional. Ela é usada para substituir a direção do texto atual.

Como comentar o código?

Para comentar o código em HTML, utilize <!-- para abrir o comentário e --> para fechar. Tudo que estiver entre essas marcas não será exibido na página. Essa prática é necessária para deixar registrado o que está sendo feito e ajudar na manutenção futura do código. Exemplo:
<!-- Este é um comentário em HTML -->
