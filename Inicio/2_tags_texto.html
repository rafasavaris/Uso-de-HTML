1) Tags de header

Tags de header em HTML são usadas para definir títulos e subtítulos em uma página, organizando o conteúdo de forma hierárquica e melhorando a semântica e a acessibilidade do site. Cabe ressaltar que elas não são autofechantes. Elas podem variar de <h1/> à <h6/>. O texto que deve aparecer como títulos ou subtítulos é escrito entre a abertura e o fechamento da tag. A tag <h1/> é indicada ser usada só uma vez, pois deve ser o título. Abaixo, estão exemplos dessas tags.

4) Como adicionar uma imagem?

Para adicionar uma imagem, basta colocar a tag <img src= alt= >, que é autofechante. O atributo src define o caminho da imagem, que pode ser uma URL, enquanto o atributo alt é uma descrição da imagem, para fins de erro no carregamento da imagem e acessibilidade. Para alterar o tamanho da imagem, podemos usar o atributo width, que define a largura da imagem, e height, o qual define a altura da imagem. Para esses dois casos, é preciso definir uma medida de tamanho (pixel ou outras). No atributo style, também é possível usar essas definições de altura e largura.

Abaixo são inseridas duas imagens, com os respectivos códigos:
<img src="./img/michael_scott.jpg" alt="Steve Carell como Michael Scott, na série 'The Office'" height="200px">
<img src="./img/pyramid_the_office_meme.jpg" alt="Meme 'é um esquema de pirâmide', da série The Office" style="height: 200px;">

5) Como adicionar links?

Para adicionar links, basta colocar a tag <a href="" target="" >nome_do_link</a>, que não é autofechante. Nessa tag, há: href, o qual informa o endereço do link; target, que define como o link será aberto (_self = abre o link na mesma página, se target não for explicitado, é determinado como _self; _blank = abre na nova página; há outros dois que serão abordados depois). É possível estilizar os links, colocando tittle, o qual coloca uma "legenda" no link, quando o mouse é passado por cima dele, e nome_do_link, é o nome visível do link.

5.1) Caracteristicas dos links

Os links podem levar para:

  • Outras abas (feitas pelo programador ou não)
  • Partes da própria página
  • Realizar contato!

Os exemplos abaixos, em ordem, abrem o Google numa nova aba, abre uma nova página html e retorna para o tópico 3 dessa página.

<a href="https://www.google.com" target="_blank" title="Link para o Google">Link para o google</a> - Link para o google

<a href="./contato.html" target="_blank">Página de contato</a> - Contato

Para retornar para um item da página atual, é preciso que um "id" seja atribuido a essa parte. Para isso, foi feito <h2 id="inicio-tags">3.2) Principais tags de formatação</h2>. Assim, o href do link é justamente esse id, acrescido de uma # na frente.

5.2) Como realizar contatos a partir de links?

Contato por e-mail:
Colocar no endereço do link (href): "mailto:email_pessoa", o que é mailto:endeco_email@arroba.com
Um exemplo é: <a href="mailto:endereco_email@arroba.com" target="_self" >nome do link >Nome do link</a>
Exemplo: Mandar email!
Contato por telefone:
Colocar no endereço do link (href): "tel:numero_telefone", isto é, tel:+559999999999
Exemplificando: <a href="tel:554899999999999" target="_self" >nome do link >Nome do link</a>
Exemplo: Telefonar para fulano
Contato por SMS:
Colocar no endereço do link (href): "sms:numero_telefone", ou seja, sms:+559999999999
Um exemplo é: <a href="sms:554899999999999" target="_self" >nome do link >Nome do link</a>
Exemplo: Envia um sms

6) Listas em HTML

Em HTML, listas são usadas para organizar e agrupar conteúdos relacionados. Existem três tipos principais de listas: listas ordenada (<ol>); lista não-ordenada (<ul>) e lista de definição (<dl>). Elas são apresentadas abaixo.

6.1) Lista ordenada

Listas ordenadas apresentam ordem, como o próprio nome indica! Para escrevê-las, basta colocar <ol>, (Order List) uma tag que não é autofechante. Para colocar cada item da lista, dentro da tag anterior, coloca-se <li> (que também não é autofechante) e dentro dela o item propriamente dito.
É possível determinar qual a ordem da lista, com o atributo style="", que pode ser 1 (números literais), i (números romanos) e a (letras), sendo que os dois últimos podem ser em maiúsculo ou minúsculo, que há diferenças. É possível determinar como a contagem começa, com o atributo start="". Um exemplo é visto abaixo, para iniciar a lista de exemplo.
Exemplo: <ol type="1" start="0"> — contagem númerica, começando em 0.

  1. item 1
  2. item 2
  3. item 3

6.2) Lista não-ordenada

Essa lista não possui ordem, são apenas tópicos. Ela é aberta com <ul> (Unorder List) e também não é autofechante. Dentro dela, também colocamos a tag <li>, para elencar os itens da lista. É possível estilizar a lista com o atributo list-style-type:"", em que se define a forma do início dos tópicos (square, circle, none, entre outras). Abaixo, um exemplo disso.
Exemplo: <ul style="list-style-type: square> — lista com quadrados de icones.

  • item 1
  • item 2
  • item 3

6.3) Lista de descrição

Essa lista é usada para apresentar termos seguidos de suas definições. Ela é aberta com a tag <dl> (Definition List) e também não é autofechante. Dentro dela, usamos a tag <dt> para indicar o termo, e a tag <dd> para descrever ou definir esse termo, sendo possível adicionar mais de uma descrição. Abaixo, um exemplo disso.
Exemplo:
<dl>
<dt>Item 1</dt>
<dd>Descrição do item 1, explicando o que ele significa ou representa.</dd>
</dl>

Item 1
Descrição do item 1, explicando o que ele significa ou representa.
Item 2
Descriçao do item 2
Descriçao 2 do item 2
