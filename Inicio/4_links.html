Links em HTML

1) Como adicionar links?

Para adicionar links, basta colocar a tag <a href="" target="" tittle="" >nome_do_link</a>, que não é autofechante. Nessa tag, há:

  • href - endereço do link;
  • target - como o link será aberto;
  • tittle - legenda do link quando o mouse é passado por cima dele;
  • nome_do_link - nome visível do link.

O link pode ser aberto de algumas formas, dependendo do atributo target:

  • _self - abre o link na mesma página (padrão);
  • _blank - abre o link numa nova página;
  • _parent - abre o link no quadro pai (útil para links embutidos);
  • _top - abre o link em todo o corpo da janela (ignora outros possíveis links abertos).

1.1) Caracteristicas dos links

Os links podem levar para:

  • Outras abas (feitas pelo programador ou não)
  • Partes da própria página
  • Realizar contato!

Os exemplos abaixos, em ordem, abrem o Google numa nova aba, abre uma nova página html e retorna para o topo dessa página.

<a href="https://www.google.com" target="_blank" title="Link para o Google">Link para o google</a> - Link para o google

<a href="./contato.html" target="_blank">Página inicial: Guia HTML</a> - Principais Conceitos HTML

Para retornar para um item da página atual, é preciso que um "id" seja atribuido a essa parte, ou seja, que tal parte tenha um atributo id associado. Para isso, foi feito:
<h2 id="#inicio">1) Como adicionar links?</h2>.
Assim, o href do link é justamente o id associado a parte que queremos acessar, acrescido de uma #.

<a href="#inicio" target="_self">Voltar para o topo da página</a> - Voltar para o topo

1.2) Como realizar contatos a partir de links?

Contato por e-mail:
Colocar no endereço do link (href): "mailto:email_pessoa", o que é: mailto:endereco_email@arroba.com
Um exemplo é: <a href="mailto:endereco_email@arroba.com" target="_self">nome do link>Nome do link</a>
Exemplo: Mandar email!
Contato por telefone:
Colocar no endereço do link (href): "tel:numero_telefone", isto é, tel:+559999999999
Exemplificando: <a href="tel:554899999999999" target="_self">nome do link>Nome do link</a>
Exemplo: Telefonar para fulano
Contato por SMS:
Colocar no endereço do link (href): "sms:numero_telefone", ou seja, sms:+559999999999
Um exemplo é: <a href="sms:554899999999999" target="_self">nome do link >Nome do link</a>
Exemplo: Envia um sms

1.3) Imagens como links:

É possível fazer com que imagens sejam links clicáveis. Para isso, juntamos as tag de link e de imagem, conforme exemplo abaixo, em que redirecionamos o Ícone do G-mail para o envio de um e-mail:
<a href="mailto:fulano@gmail.com">
<img src="./img/gmail_icon.png" alt="Ícone do G-mail" style="height: 50px;">
</a>
