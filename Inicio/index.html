Guia de conceitos para o corpo HTML!

Este é um guia de conceitos iniciais para o corpo do html, isto é, o corpo da tag <body></body>. A ideia desse arquivo é funcionar como um "dicionário", que possa ser visitado quando alguma dúvida surgir. Portanto, sinta-se à vontade para usar!

1) O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas na web. Ele serve para estruturar o conteúdo, organizando textos, imagens, links, tabelas, vídeos e outros elementos de forma que os navegadores possam exibir corretamente. Basicamente, o HTML define o que vai aparecer em uma página e como esses elementos estão organizados

2) O que são tags em HTML?

Tags em HTML são usadas para estruturar o conteúdo de uma página. A maioria das tags possui uma abertura <tag> e um fechamento </tag>, como por exemplo: <p>Este é um parágrafo</p>.

2.1) Tags autofechantes

Algumas tags são autofechantes, ou seja, não precisam de uma tag de fechamento separada. Elas são encerradas dentro da própria abertura. Exemplos:

  • <br /> — quebra de linha
  • <img src="imagem.jpg" alt="Descrição" /> — imagem
  • <input type="text" /> — campo de entrada
  • <meta charset="UTF-8" /> — metadados
  • <hr /> — linha horizontal

3) Principais tags e exemplos:

3.1) Tags de header

Tags de header em HTML são usadas para definir títulos e subtítulos em uma página, organizando o conteúdo de forma hierárquica e melhorando a semântica e a acessibilidade do site. Cabe ressaltar que elas não são autofechantes. Elas podem variar de <h1/> à <h6/>. O texto que deve aparecer como títulos ou subtítulos é escrito entre a abertura e o fechamento da tag. A tag <h1/> é indicada ser usada só uma vez, pois deve ser o título. Abaixo, estão exemplos dessas tags.

4) Como adicionar uma imagem?

Para adicionar uma imagem, basta colocar a tag <img src= alt= >, que é autofechante. O atributo src define o caminho da imagem, que pode ser uma URL, enquanto o atributo alt é uma descrição da imagem, para fins de erro no carregamento da imagem e acessibilidade. Para alterar o tamanho da imagem, podemos usar o atributo width, que define a largura da imagem, e height, o qual define a altura da imagem. Para esses dois casos, é preciso definir uma medida de tamanho (pixel ou outras). No atributo style, também é possível usar essas definições de altura e largura.

Abaixo são inseridas duas imagens, com os respectivos códigos:
<img src="./img/michael_scott.jpg" alt="Steve Carell como Michael Scott, na série 'The Office'" height="200px">
<img src="./img/pyramid_the_office_meme.jpg" alt="Meme 'é um esquema de pirâmide', da série The Office" style="height: 200px;">

5) Como adicionar links?

Para adicionar links, basta colocar a tag <a href="" target="" >nome do link >Nome do link</a>, que não é autofechante. Nessa tag, há: href, o qual informa o endereço do link; target, que define como o link será aberto (_self = abre o link na mesma página, se target não for explicitado, é determinado como _self; _blank = abre na nova página). É possível estilizar os links, colocando tittle, o qual coloca uma "legenda" no link, quando o mouse é passado por cima dele, e Nome do link, que é o texto escrito antre a abertura e o fechamento da tag, é o nome visível do link.

5.1) Caracteristicas dos links

Os links podem levar para:

  • Outras abas (feitas pelo programador ou não)
  • Partes da própria página
  • Realizar contato!

Os exemplos abaixos, em ordem, abrem o Google numa nova aba, abre uma nova página html e retorna para o tópico 3 dessa página.

<a href="https://www.google.com" target="_blank" title="Link para o Google">Link para o google</a> - Link para o google

<a href="./contato.html" target="_blank">Página de contato</a> - Contato

5.2) Como realizar contatos a partir de links?

Contato por e-mail:
Colocar no endereço do link (href): mailto:endeco_email@arroba.com
Um exemplo é: <a href="mailto:endereco_email@arroba.com" target="_self" >nome do link >Nome do link</a>
Exemplo: Mandar email!
Contato por telefone:
Colocar no endereço do link (href): tel:+559999999999
Exemplificando: <a href="tel:554899999999999" target="_self" >nome do link >Nome do link</a>
Exemplo: Telefonar para fulano
Contato por SMS:
Colocar no endereço do link (href): sms:+559999999999
Um exemplo é: <a href="sms:554899999999999" target="_self" >nome do link >Nome do link</a>
Exemplo: Envia um sms
