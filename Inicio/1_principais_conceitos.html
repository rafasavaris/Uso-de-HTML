Guia de conceitos para o corpo HTML!

Este é um guia de conceitos iniciais para o HTML, focando no corpo dele, isto é, o corpo da tag <body></body>. A ideia desse arquivo é funcionar como um "dicionário", que possa ser visitado quando alguma dúvida surgir. Portanto, sinta-se à vontade para usar! Será algo simples, pois o foco são as principais tags e conceitos HTML. Conforme o conteúdo é aprofundado, melhor os documentos se tornam.

1) O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas na web. Ele serve para estruturar o conteúdo, organizando textos, imagens, links, tabelas, vídeos e outros elementos de forma que os navegadores possam exibir corretamente. Basicamente, o HTML define o que vai aparecer em uma página e como esses elementos estão organizados. Os elementos HTML, no caso, são uma tag inicial, algum conteúdo e a tag final (quando houver).

2) O que são tags em HTML?

Tags em HTML são usadas para estruturar o conteúdo de uma página. A maioria das tags possui uma abertura <tag> e um fechamento </tag>, como por exemplo: <p>Este é um parágrafo</p>.

2.1) Tags autofechantes

Algumas tags são autofechantes, ou seja, não precisam de uma tag de fechamento separada. Elas são encerradas dentro da própria abertura. Exemplos:

  • <br> — quebra de linha;
  • <img src="imagem.jpg" alt="Descrição"> — imagem;
  • <input type="text"> — campo de entrada;
  • <meta charset="UTF-8"> — metadados;
  • <hr> — linha horizontal.

3) O que são atributos?

Atributos são caracteristícas específicas de cada tag. Elas determinam como a tag deve se comportar ou como deve ser exibida. Os atributos são colocados dentro da tag de abertura, após o nome da tag, e são compostos por um nome e um valor, separados por um sinal de igual. Por exemplo, na tag <img src="imagem.jpg" alt="Descrição" />, src é o nome do atributo que define o caminho da imagem, e alt é o atributo que fornece uma descrição alternativa para a imagem. Alguns atributos comuns incluem:

  • id: identifica de forma única um elemento na página;
  • class: permite agrupar elementos para aplicar estilos ou scripts;
  • style: define estilos CSS.

4) Principais elementos HTML:

Aqui estão alguns dos elementos HTML que serão abordadas nas próximas páginas:

  • <br> — quebra de linha
  • <hr> — linha horizontal
  • <abbr> - define uma abreviação/acrônimo, o atributo tittle pode mostrar a descrição quando o mouse passa por cima;
  • <style> — definir informações de estilo para uma única página HTML;
  • <script> — definir JavaScript do lado do cliente;
  • <h1>...</h1> até <h6>...</h6> — títulos e subtítulos
  • <p>...</p> — parágrafo de texto
  • <a href="...">...</a> — define um hiperlink (link);
  • <img src="..." alt="..." /> — imagem
  • <ul>, <ol>, <li> — listas não ordenadas, ordenadas e itens de lista
  • <table>, <tr>, <td>, <th> — tabelas
  • <div>...</div> — divisão de bloco para agrupar conteúdo
  • <span>...</span> — agrupamento em linha de conteúdo
  • <form>...</form> — formulário para entrada de dados
  • <input>, <button>, <label> — elementos de formulário
  • <strong>...</strong>, <em>...</em> — destaque e ênfase em textos

5) Estrutura inicial de um arquivo HTML

A estrutura de um arquivo HTML começa com uma declaração !DOCTYPE HTML, que identifica para os navegadores qual o tipo de documento eles devem esperar. Em seguida, deve ser incluído o elemento <html>. Ele é o elemento raíz de uma página HTML, ou seja,o documento começa com a abertura dele e termina com o seu fechamento. Um atributo essencial desse elemento é lang, que significa language, isto é, o idioma da página. Nesse guia, o idioma é PT-BR (português brasileiro).

Dentro do elemento <html> é adicionado os outros elementos, sendo <head> o primeiro. Ele é um contêiner de metadados (dados sobre dados), ou seja, são dados sobre o documento HTML, os quais não são exibidos. Esses metadados normalmente são:

  • <tittle> - define o título do documento, entre sua abertura e fechamento;
  • <meta> - especifica o conjunto de caracteres, descrição da página, autor do documento e configurações gerais;
  • <link> - define a relação do documento atual e um recurso externo (páginas de estilo, favicons).
  • <base> - especifica a URL base de todas da página, isto é, onde elas serão abertas (atributo href) ou como elas serão abertas (argumento _target).

Após isso, é aberto o elemento <body> e, entre ele, é definido o corpo do documento, onde é adicionado todo o conteúdo da página. Essa é basicamente a estrutura padrão da página, o que muda é o que é posto dentro. É comum adicionar no fim da página o elemento <address> que define informações de contato do autor.
