Guia de conceitos para o corpo HTML!

Este é um guia de conceitos iniciais para o corpo do html, isto é, o corpo da tag <body></body>. A ideia desse arquivo é funcionar como um "dicionário", que possa ser visitado quando alguma dúvida surgir. Portanto, sinta-se à vontade para usar!
A ideia deste arquivo é ser simples, pois o foco são as principais tags e conceitos HTML.

1) O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas na web. Ele serve para estruturar o conteúdo, organizando textos, imagens, links, tabelas, vídeos e outros elementos de forma que os navegadores possam exibir corretamente. Basicamente, o HTML define o que vai aparecer em uma página e como esses elementos estão organizados

2) O que são tags em HTML?

Tags em HTML são usadas para estruturar o conteúdo de uma página. A maioria das tags possui uma abertura <tag> e um fechamento </tag>, como por exemplo: <p>Este é um parágrafo</p>.

2.1) Tags autofechantes

Algumas tags são autofechantes, ou seja, não precisam de uma tag de fechamento separada. Elas são encerradas dentro da própria abertura. Exemplos:

  • <br /> — quebra de linha
  • <img src="imagem.jpg" alt="Descrição" /> — imagem
  • <input type="text" /> — campo de entrada
  • <meta charset="UTF-8" /> — metadados
  • <hr /> — linha horizontal

3) O que são atributos?

Atributos são caracteristícas específicas de cada tag. Elas determinam como a tag deve se comportar ou como deve ser exibida. Os atributos são colocados dentro da tag de abertura, após o nome da tag, e são compostos por um nome e um valor, separados por um sinal de igual. Por exemplo, na tag <img src="imagem.jpg" alt="Descrição" />, src é o nome do atributo que define o caminho da imagem, e alt é o atributo que fornece uma descrição alternativa para a imagem. Alguns atributos comuns incluem:

  • id: identifica de forma única um elemento na página.
  • class: permite agrupar elementos para aplicar estilos ou scripts.
  • style: define estilos CSS,

3) Principais tags e exemplos:

Aqui estão algumas das principais tags HTML e seus usos:

  • <h1>...</h1> até <h6>...</h6> — Títulos e subtítulos
  • <p>...</p> — Parágrafo de texto
  • <a href="...">...</a> — Link para outra página ou recurso
  • <img src="..." alt="..." /> — Imagem
  • <ul>, <ol>, <li> — Listas não ordenadas, ordenadas e itens de lista
  • <table>, <tr>, <td>, <th> — Tabelas
  • <div>...</div> — Divisão de bloco para agrupar conteúdo
  • <span>...</span> — Agrupamento em linha de conteúdo
  • <form>...</form> — Formulário para entrada de dados
  • <input>, <button>, <label> — Elementos de formulário
  • <br /> — Quebra de linha
  • <hr /> — Linha horizontal
  • <strong>...</strong>, <em>...</em> — Destaque e ênfase em textos
