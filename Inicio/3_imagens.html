Imagens em HTML

1) Como adicionar uma imagem?

Para adicionar uma imagem, basta colocar a tag <img src= alt= >, que é autofechante. O atributo src define o caminho da imagem, que pode ser uma URL, enquanto o atributo alt é uma descrição da imagem, para fins de erro no carregamento da imagem e acessibilidade. Para alterar o tamanho da imagem, podemos usar o atributo width, que define a largura da imagem, e height, o qual define a altura da imagem. Para esses dois casos, é preciso definir uma medida de tamanho (pixel ou outras). No atributo style, também é possível usar essas definições de altura e largura. Essa tag é inline, ou seja, permite que seja colocada dentro de outra tag, apesar de não ser recomendado. Nesse guia, será posto fora de outras tags, a não ser que seja necessário.

Abaixo são inseridas duas imagens, com os respectivos códigos:
<img src="./img/michael_scott.jpg" alt="Steve Carell como Michael Scott, na série 'The Office'" height="200px">
<img src="./img/pyramid_the_office_meme.jpg" alt="Meme 'é um esquema de pirâmide', da série The Office" style="height: 200px;">

2) O que é um favicon?

Favicon é uma pequena imagem (ícone) exibida na guia do navegador ao lado do título da página. Os tipos aceitos são: ico, png, jpeg, gif e svg. Para adicionar um favicon, usamos a tag <link rel="icon" href="caminho_do_favicon">, que não é autofechante. O atributo rel define o tipo de link, que é "icon"; o atributo href define o caminho do favicon. Essa tag deve ser colocada dentro da tag <head></head>.

3) Tag <figure>

O atributo <figure> é usada para agrupar conteúdo de mídia independente como imagens, gráficos, diagramas, trechos de código e etc., junto com sua legenda descritiva. Geralmente, é acompanhada do atributo <figcaption> que fornece a legenda/explicação. O uso dessas tags é importante para acessibilidade (indica que o conteúdo está relacionado ao texto para leitores de tela e navegadores) e indica que o texto é independente, ou seja, pode ser movido para outra posição sem perder sentido. Um exemplo é visto abaixo:

<figure>
 <img src="./img/icon_gato.jpg" alt="Ícone de um gato preto" width="100" height="100">
 <figcaption>Ícone de um gato preto.</figcaption>
</figure>

4) Tag <area>

A tag <area> define uma área dentro de um mapa de imagem (uma imagem com áreas clicláveis). Os elementos <area> são sempre aninhados dentro de uma tag <map>. Para tornar uma imagem cliclável, basta adicionar na tag <img> o atributo usemap, em que seu valor é igual ao atributo name da tag <map> acrescido de uma # antes.

4.1) Atributos da tag <area>

- shape - define o formato da área. Pode ser:
 rect - X1, Y1, X2, Y2;
 circle - CENTROX, CENTROY, RAIO;
 poly - várias coordenadas;
 default - toda a imagem.
- coords - define as coordenadas da área, relativas ao canto superior esquerdo da imagem;
- href - link para onde a área aponta;
- alt - texto alternativo para acessibilidade;
- target - onde abrir o link.
Hoje em dia, devido a complexidade para responsividade, não é tão utilizado. Um exemplo é visto abaixo:

<img src="./img/michael_scott_thats_what_she_said.png" alt="Meme The Office 'That's what she said'" height="200" usemap="#mapa1">
<map name="mapa1">
 <area shape="circle" coords="100, 100, 50" href="./2_tags_texto.html" alt="Voltar para página 2." target="_self">
 <area shape="rect" coords="0, 0, 50, 50" href="./4_links.html" alt="Próxima página." target="_self">
</map>
