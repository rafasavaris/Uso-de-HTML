Como montar tabelas em HTML?

Tabelas são usadas para organizar informações em linhas e colunas. Em HTML, as tabelas são criadas a partir da tag <table>, que não é autofechante.

Índice

  • Estrutura básica de uma tabela
  • Exemplo de tabela simples
  • Exemplo de tabela estilizada

1) Estrutura básica de uma tabela

A estrutura básica de uma tabela é composta por várias tags, que são:

  • <table> - define o início e o final da tebla;
  • <tr> - define uma linha na tabela;
  • <th> - define um cabeçalho de coluna;
  • <td> - define uma célula de dados;
  • <tbody> - define o corpo da tabela, onde ficam as linhas de dados;
  • <thead> - define o cabeçalho da tabela, onde ficam os títulos das colunas;
  • <tfoot> - define o rodapé da tabela, onde ficam informações adicionais ou totais;
  • <caption> - fornece um título/legenda para uma tabela HTML. deve ser a primeira tag dentro de <table>.

1.1) Porque usar <thead>, <tbody> e <tfoot>?

Caso você tenha reparado, as tags <thead>, <tbody> e <tfoot> não são obrigatórias para a criação de uma tabela. Elas são usadas para melhorar a semântica e a acessibilidade da tabela, além de facilitar a estilização com CSS. A tag <thead> é usada para agrupar os cabeçalhos das colunas, a tag <tbody> é usada para agrupar as linhas de dados e a tag <tfoot> é usada para agrupar informações adicionais ou totais. Cabe ressaltar que não é um elemento inline.

2) Exemplo de tabela

Abaixo, um exemplo de tabela simples, com três colunas (Nome, Função e Idade) e três linhas de dados. A tabela utiliza as tags <thead> e <tbody> para organizar o conteúdo, mas não apresenta <tfoot>.

Fulano B Profissão B Idade B
Fulano C Profissão C Idade C

<table>
 <thead>
  <tr>
   <th>Nome</th>
   <th>Função</th>
   <th>Idade</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <tdFulano>
   <td>Profissão A</td>
   <td>Idade A</td>
  </tr>
  <tr>
   <td>Fulano B</td>
   <td>Profissão B</td>
   <td>Idade B</td>
  </tr>
  <tr>
   <td>Fulano C</td>
   <td>Profissão C</td>
   <td>Idade C</td>
  </tr>
 </tbody>
</table>

3) Como estilizar tabelas?

A tabela mostrada anteriormente é simples e sem estilo. Para melhorar a aparência das tabelas, é possível utilizar atributos para adicionar cores, bordas, espaçamentos e outros efeitos visuais em HTML puro.

3.1) Tag <colgroup>

A tag <colgroup> é usada para agrupar uma ou mais colunas dentro de uma tabela, permitindo aplicar estilos ou atributos de forma coletiva a elas. Ela deve ficar logo após a tag <table> e antes de <thead> ou <tbody>. Dentro de <colgroup>, usamos a tag <col> para definir cada coluna individualmente. A tag <col> é autofechante e pode receber atributos como span (número de colunas que o estilo abrange), width (largura da coluna) e bgcolor (cor de fundo da coluna). Um exemplo é:

<table>
 <colgroup>
   <col span="1" width="100" bgcolor="lightblue">
   <col span="2" width="100" bgcolor="lightgreen">
 </colgroup>
 <thead>
  <tr>
   <th>Nome</th>
   <th>Função</th>
   <th>Idade</th>
   </tr>
 </thead>
 <tbody>
   <tr>
    <tdFulano>
    <td>Profissão A</td>
    <td>Idade A</td>
  </tr>
  <tr>
   <td>Fulano B</td>
   <td>Profissão B</td>
   <td>Idade B</td>
  </tr>
  <tr>
   <td>Fulano C</td>
   <td>Profissão C</td>
   <td>Idade C</td>
  </tr>
 </tbody>
</table>

3.2) Principais atributos para estilizar tabelas

Alguns dos principais atributos que podem ser usados para estilizar tabelas são:

  • border: define a largura da borda da tabela. Apenas números inteiros;
  • cellpadding: define o espaçamento interno entre o conteúdo da célula e a borda;
  • cellspacing: define o espaçamento entre as células da tabela;
  • width: define a largura da tabela;
  • height: define a altura da tabela;
  • bgcolor: define a cor de fundo da tabela ou das células;
  • align: define o alinhamento do conteúdo dentro das células (left, center, right);
  • valign: define o alinhamento vertical do conteúdo dentro das células (top, middle, bottom).

Abaixo, um exemplo de tabela estilizada com bordas, cores e espaçamentos. O código HTML é o mesmo da tabela anterior, mas com a adição dos seguintes atributos na tag <table>: border="1" width="300" height="300" bgcolor="pink" align="center" cellpadding="10" cellspacing="5". Esses atributos definem a largura da borda, a largura e altura da tabela, a cor de fundo, o alinhamento da tabela e o espaçamento interno e entre as células.

4) Como expandir células?

Para quem já é acostumado a mexer com tabelas, sabe que é possível expandir células para elas ocuparem mais colunas ou mais linhas. Em HTML isso também é possível com os atributos rowspan e colspan. Os valores desses atributos são o número de células a serem expandidas. O exemplo abaixo utiliza o mesmo código do primeiro exemplo feito, mas com a célula Profissão A ocupando duas linhas (de forma que Profissão B não existe mais) e Profissão C ocupa duas colunas, sendo possível perceber que, então, Idade C é colocada ao lado (numa quarta coluna).
