1) O que é HTML semântico?

HTML semântico é uma abordagem de marcação que utiliza elementos HTML de forma a transmitir o significado do conteúdo. Em vez de usar tags genéricas, como `div` e `span`, o HTML semântico utiliza tags específicas, como `article`, `section`, `header`, `footer`, entre outras, para descrever a estrutura e o propósito do conteúdo.

1.1) Vantagens do HTML semântico

  • Acessibilidade: melhora a navegação popr tecnologias asssistivas;
  • Otimização para motores de busca: algoritmos podem entender melhor a estrutura e o conteúdo, priorizando no ranking de buscas;
  • Manutenção: códigos semanticamente estruturados são mais limpos e organizados, facilitando a manutenção;
  • Compatibilidade: garante que o conteúdo da página será exibido igual em diferentes navegadores.

2) Exemplos de tags utilizadas no HTML semântico

Abaixo estão algumas das principais tags semânticas do HTML5 e suas descrições. É importante ressaltar que essas tags não são autofechantes.

2.1) <header>

A tag <header> é usada para definir o cabeçalho de uma página ou seção. Ela geralmente contém o título, logotipo e navegação principal.

2.2) <nav>

A tag <nav> é utilizada para blocos de navegação, como menus principais ou de rodapé, estruturando links externos/internos de forma organizada.

2.3) <main>

A tag <main> é usada para definir o conteúdo principal de uma página. Ela deve conter o conteúdo mais relevante e único, excluindo cabeçalhos, rodapés e barras laterais.

2.4) <article>

A tag <article> é usada para conteúdos independentes e autossuficientes.

2.5) <section>

A tag <section> é usada para definir seções dentro de um documento, agrupando conteúdo relacionado.

2.6) <aside>

A tag <aside> é usada para definir conteúdo relacionado, mas não essencial, ao conteúdo principal da página. É frequentemente usado para barras laterais, citações ou links adicionais.

2.7) <footer>

A tag <footer> é usada para definir o rodapé de uma página ou seção. Ela geralmente contém informações de copyright, links de contato e outras informações complementares.
