Texto em HTML

1) Header

Elementos de header em HTML são usados para definir títulos e subtítulos em uma página, organizando o conteúdo de forma hierárquica e melhorando a semântica e a acessibilidade do site. Cabe ressaltar que elas não são autofechantes. Elas podem variar de <h1/> à <h6/>. O texto que deve aparecer como títulos ou subtítulos é escrito entre a abertura e o fechamento da tag. A tag <h1/> é indicada ser usada só uma vez, pois deve ser o título. Abaixo, estão exemplos:

Esta é uma tag h1

Esta é uma tag h2

Esta é uma tag h3

Esta é uma tag h4

Esta é uma tag h5
Esta é uma tag h6

2) Parágrafos em HTML

O elemento <p> é utilizado para definir parágrafos em HTML. Ele delimita blocos de texto, separando visualmente e semanticamente as ideias no conteúdo da página. Todo texto colocado entre <p> e </p> será exibido como um novo parágrafo, com espaçamento automático antes e depois. É uma das tags mais usadas para estruturar textos em páginas web. Se desejado quebrar uma linha, deve ser usada a tag <br>.

3) Principais tags de formatação

As tags de formatação em HTML são usadas para alterar a aparência e o estilo do texto, sem necessariamente mudar sua estrutura semântica. Abaixo, são apresentadas algumas dessas tags, sendo que nenhum é autofechante.

- Tag <b> — Serve para colocar o texto em negrito
- Tag <strong> — Também coloca o texto em negrito, mas dá mais ênfase em buscas em aplicativos de textos/internet
- Tag <i> — Coloca o texto em itálico
- Tag <i> — Coloca o texto em itálico.
- Tag <em> — Também coloca o texto em itálico, mas com ênfase para aplicativos de texto/internet
- Tag <mark> — Marca o texto como um marca-texto
- Tag <small> — Diminui o tamanho do texto
- Tag <sup> — Coloca o texto superior, dessa forma
- Tag <sub> — Coloca o texto inferior, assim
- Tag <del> — "Risca" o texto, dessa maneira
- Tag <ins> — Define um texto que foi inserido num documento. Normalmente, o navegador sublinha, assim: texto
- Tag <code> — Coloca o texto como monoespaçado, isto é, letra em formato de código (tendo esse significado semânticamente), como em: exemplo!. Deve ser escrito dentro de <p> apenas para pequenos códigos.
- Tag <blockquote> — Define uma seção citada de outra fonte. Os navegadores geralmente recuam esses elementos, além de que eles são elementos independentes, usados fora de paragráfos, como em seguida:

Às vezes a vida é um filme e eu sou uma atriz no Vale do Silício. Rimando como se a minha vida dependesse disso. - Extraordinária, Ebony, 2025.

- Tag <q> — Define uma citação curta. Navegadores normalmente inserem aspas ao redor, como visto em seguida:
Me sinto extraordinária, dona de uma mente brilhante. - Extraordinária, Ebony - 2025
- Tag <address> — Define as informações de contato do autor/proprietário de um documento ou artigo. O texto no elemento geralmente é renderizado em itálico, e os navegadores sempre adicionam uma quebra de linha antes e depois desse elemento.
- Tag <cite> — Define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura etc.). O texto geralmente é renderizado em itálico. Por exemplo: O capital, livro de Karl Marx.
- Tag <bdo> — BDO significa substituição bidirecional. Ela é usada para substituir a direção do texto atual. Para isso, essa tag possui o atributo dir, que pode ser preenchido como ltr (left to right - esquerda para direita) ou rtl (right to left - direita para esquerda). Isso é mostrado abaixo, na ordem apresentada:

<bdo dir="ltr">Texto da Esquerda Para Direita (normal)</bdo>
<bdo dir="rtl">Texto da Direita Para Esquerda (invertido)</bdo>

Texto da Esquerda para Direita (normal)
Texto da Direita para Esquerda (invertido)

- Tag <pre> — Exibe o texto exatamente como ele foi escrito no código-fonte, preservando indentação, vários espaços e quebras de linha. Ele não deve ser escrito dentro de <p> pois é um bloco, além de ter fonte monoespaçada. O uso dele é visto em seguida: 

        Exemplo                  de             (quebra de linha agora)
        tag            <pre>

4) Como comentar o código?

Outra forma de texto, porém, em relação ao código são os comentários! Para comentar o código em HTML, utilize <!-- para abrir o comentário e --> para fechar. Tudo que estiver entre essas marcas não será exibido na página. Essa prática é necessária para deixar registrado o que está sendo feito e ajudar na manutenção futura do código. Exemplo:
<!-- Este é um comentário em HTML -->

5) Entidades em HTML

Em HTML, alguns caracteres tem significados reservados ou não podem ser digitadas no código-fonte. Assim, as entidades são códigos especiais para escrever tais caracteres. Elas são escritas com & + código + ;. Algumas das entidades estão listadas abaixo:

- Caractere "<" - código: &lt;
- Caractere ">" - código: &gt;
- Caractere "&" - código: &amp;
- Caractere """ - código: &quot;
- Caractere "'" - código: &apos;
- Caractere "©" - código: &copy;
- Caractere "®" - código: &reg;
- Caractere "™" - código: &trade;
- Espaço não separável: " " - código: &nbsp;
- Espaço médio: " " - código: &ensp;
- Espaço largo: " " - código: &emsp;

6) Detalhes adicionais sob demanda

É possível criar um espaço dentro da página em que o usuário pode abrir e fechar o conteúdo apenas clicando. Para isso, é usado a tag <details> que cria um widget interativo, o qual pode ser fechado e aberto pelo usuário, sendo fechado o seu padrão. Qualquer tipo de conteúdo pode ser posto nessa tag. Além disso, a tag <summary> serve como um cabeçalho para essa tag. Um exemplo de uso pode ser visto abaixo:

<details>
 <summary>Veja o texto escondido!</summary>
 Criciúma Esporte Clube campeão moral da Libertadores de 1992.
</details>

O que gera:

Veja o texto escondido! Criciúma Esporte Clube campeão moral da Libertadores de 1992.

7) Tag <time>

Às vezes, é necessário representar datas, horários ou períodos de tempo de forma que tanto máquina quanto pessoas entendam. Para isso, usa-se o elemento <time>. O que aparece entre a tag é o que o usuário vê, enquanto o atributo datatime contém a versão padronizada para máquinas entenderem. O datetime pode ter os seguintes formatos:

Data: YYYY-MM-DD
Hora: HH:MM
Data e hora: YYYY-MM-DDTHH:MM
Com fuso horário: 2025-08-09T14:30-03:00
Mês e ano: 2025-08
Semana: 2025-W32
Duração: P2DT3H

Um exemplo de uso é: < time datetime="2025-08-10";>10 de agosto de 2025</time>
Essa tag pode ser usada dentro de um parágrafo, o que gera, segundo o exemplo acima:
10 de agosto de 2025
