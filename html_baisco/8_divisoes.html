Formas de seccionar partes de um documento HTML

Neste arquivo, será apresentado como seccionar (dividir) partes de um documento HTML utilizando as tags div, span e iframe. Essas tags são fundamentais para organizar o conteúdo e melhorar a estrutura do documento, facilitando a leitura e a manutenção do código.

1) Introdução às tags de seccionamento

As tags de seccionamento são essenciais para organizar o conteúdo de uma página HTML. Elas permitem que o desenvolvedor agrupe elementos relacionados, facilitando a leitura e a compreensão do documento tanto para usuários quanto para motores de busca.
Neste documento, vamos explorar três tags importantes: <div>, <span> e <iframe>, sendo que nenhuma é autofechante.

2) Tag <div>

A Tag <div> é um contêiner genérico, usado para agrupar outros elementos HTML. É amplamente utilizada para aplicar estilos CSS e scripts JavaScript a um bloco. O atributo CSS é: style="background-color: darkmagenta; color: white; padding: 20px;.

A Tag <span> é um contêiner em linha, usado para agrupar partes de texto ou outros elementos dentro de um bloco. 4) Tag <iframe>

A Tag <iframe> é um "quadro"embutido. Ela é usada para incorporar outro documento HTML dentro de uma página. É útil para exibir conteúdo externo, como vídeos, mapas ou outros sites. O exemplo abaixo mostra como usar a tag <iframe> para incorporar um álbum do Spotify.
<iframe data-testid="embed-iframe" style="border-radius:12px" src="https://open.spotify.com/embed/album/5o7rJ8g9O5ltC3QJeTfvjD?utm_source=generator" width="100%" height="352" frameBorder="0" allowfullscreen="" allow="autoplay; clipboard-write; encrypted-media; fullscreen; picture-in-picture" loading="lazy"></iframe> Nessa tag, o atributo:

  • data-testid="embed-iframe" - é usado para identificar o iframe para testes automatizados.
  • title="Spotify Album KM2 - Ebony" - fornece uma descrição do conteúdo do iframe, importante para acessibilidade.
  • style="border-radius:12px" - define o raio de borda do quadro, dando um efeito arredondado.
  • src="https://open.spotify.com/embed/album/5o7rJ8g9O5ltC3QJeTfvjD?utm_source=generator" - especifica a URL do conteúdo a ser incorporado.
  • width="100%" height="352" - define a largura e altura do quadro.
  • frameBorder="0" - remove a borda do quadro.
  • allowfullscreen="" - permite que o conteúdo incorporado seja exibido em tela cheia.
  • allow="autoplay; clipboard-write; encrypted-media; fullscreen; picture-in-picture" - especifica as permissões para o conteúdo incorporado, como reprodução automática, escrita na área de transferência, reprodução de mídia protegida, tela cheia e vídeo flutuante.
  • loading="lazy" - indica que o carregamento do iframe será feito quando estiver próximo de aparecer na tela, melhorando o desempenho da página.

Normalmente, para o uso de <iframe> é necessário apenas o uso dos atribudos src, width e height. Em muitas mídias, como o Spotify e o Youtube, o código de incorporação já vem pronto para ser copiado e colado.
